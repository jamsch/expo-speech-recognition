Add support for Expo SDK 56. From SDK 56, versioning for expo-speech-recognition has changed to be in line with Expo versioning. So this means for Expo 56, you can now install expo-speech-recognition ^56.0.0 and in the future, install 57.0.0 when SDK 57 comes out.

With this version, the minimum iOS version has also been increased from 13.4 to 16.4.
